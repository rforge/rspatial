Downloading sp and related projects

There are two methods for obtaining sp or related projects:
  • Packages should be obtained from CRAN (by using install.packages(), or throught the menu: install packages from CRAN); this is the preferred way, but if you need as yet unreleased functionality, try the CVS tree
  • using CVS tree, by using anonymous cvs, e.g. by: 
    cvs -d:pserver:anonymous@r-spatial.cvs.sourceforge.net:/cvsroot/r-spatial login
cvs -z3 -d:pserver:anonymous@r-spatial.cvs.sourceforge.net:/cvsroot/r-spatial co -P sp
    where you can give a return on the passwd prompt. See this linl for further details. For other modules, replace sp with e.g. spgrass6. Click here and follow More -- Code -- CVS -- Browse the CVS Tree to satisfy your curiosity about what else is there.
